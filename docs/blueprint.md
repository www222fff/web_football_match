# **App Name**: 绿茵助手

## Core Features:

- 赛事赛程: 展示完整的足球赛事赛程安排，包括比赛时间、参赛队伍、比赛场地等。
- 比赛结果: 记录并展示已完成比赛的结果，包括比分、进球球员、红黄牌等信息。
- 球队信息: 展示参赛球队的详细信息，包括球队logo、球队成员、教练信息等。
- 实时比分: 提供比赛的实时比分更新，让用户可以及时了解比赛进展。
- 积分榜: 根据比赛结果自动更新积分榜，展示各队的排名情况。
- 球员资料: 收录球员的技术统计，用AI tool做比对，以便对球员的水平进行量化和评价
- 个性化推荐: 根据用户的喜好和关注，推荐相关的比赛和球队信息。

## Style Guidelines:

- 主色调：深绿色 (#1B5E20)，象征足球场的草地，沉稳而专业。
- 背景色：浅灰色 (#F5F5F5)，简洁干净，突出内容。
- 强调色：橙色 (#FF9800)，用于突出关键信息和操作按钮。
- 字体：'PT Sans'，一种现代的、具有可读性的非衬线字体，适用于标题和正文。
- 使用简洁的线性图标，例如足球、球场、奖杯等，增强视觉效果。
- 采用卡片式布局，信息模块化展示，方便用户快速浏览。
- 添加流畅的过渡动画，例如页面切换、数据加载等，提升用户体验。
